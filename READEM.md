NetScan CLI 🔍

A simple and fast command-line port scanner built using Bash and PowerShell.

Features

- Scan multiple ports
- Logging support
- Cross-platform (Linux & Windows)
- Easy CLI usage

Usage

Bash

./netscan.sh --target 8.8.8.8 --ports 22,80,443

PowerShell

.\NetScan.ps1 -target 8.8.8.8 -ports 22,80,443

Output

- Displays open/closed ports
- Saves logs to file
